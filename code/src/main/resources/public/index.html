> ABOUT THE PROJECT
root@jitor:~$ cat README.md

Jitor is a web server developed in Java with a focus on simplicity, full control, and deep understanding of how HTTP servers work.

Inspired by the early days of the internet, the project demonstrates how requests are processed from raw sockets to the final response sent to the client.

> HOW IT WORKS
root@jitor:~$ ./jitor --verbose

Jitor operates directly on TCP sockets, manually parsing HTTP requests.

  • Listens for connections on a defined port
  • Processes HTTP requests (GET, POST, etc.)
  • Sends responses with headers and content
  • Handles multiple connections using threads
